## Getting started

### Clone repository

```shell
git clone https://github.com/leesean5150/ReactNodeProject.git
```

### Install dependencies

```shell
npm install
```

### Run Client

```shell
cd client
npm start
```

### Run Server

```shell
cd server
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view frontend in the browser.
